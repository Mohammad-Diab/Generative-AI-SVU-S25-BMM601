# الذكاء الاصطناعي التوليدي
### Generative Artificial Intelligence

An Arabic RTL interactive slideshow (13 slides) covering Generative AI — presented for ITE_BMM601 · SVU · Spring 2025.

---

## Contents

- What is Generative AI and how it works
- Diffusion models, GANs, LLMs, image / audio / video generation
- Development tools and frameworks
- Scale, capabilities, and limitations
- AI ethics and societal impact
- Interactive demos, quizzes, and a live AI timeline

## Tech Stack

- Pure HTML / CSS / JavaScript — no framework, no build step
- Google Fonts: Changa + IBM Plex Sans Arabic
- Animated SVG logo with CSS keyframes
- Dual-canvas system: neural-network background + custom cursor
- Deep Dive overlay with per-topic expanded content

## File Structure

```
index.html        — 13-slide deck (slides 00–12)
css/style.css     — all styles, single file
js/main.js        — all interactivity, single IIFE
img/              — logo and media assets
```

## Usage

Open `index.html` directly in a browser, or use VS Code Live Server on port 5500.

> Frontend-only — no backend, no API calls. All demos are simulations running in the browser.
